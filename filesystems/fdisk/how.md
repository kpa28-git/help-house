
## Enter fdisk shell for device `<d>`
`sudo fdisk /dev/<d>`

## List commands
`fdisk> m`

## Delete Partition
`fdisk> d`

## Create GPT Partition Table (Overwrites Existing)
`fdisk> g`

## Create New Partition
`fdisk> n`

## Print Partition Table
`fdisk> p`

## Print Unpartitioned Space
`fdisk> F`

## Print Partition Information
`fdisk> i`

## Write and Quit
`fdisk> w`

## Quit Without Saving
`fdisk> q`
